# My Entry for Enrolla Coding Challenge

## How to run the application

### If you have docker installed

On /docker directory, run `docker compose -up`

#### If you have make installed (much easier)

First you will go to the /docker directory and run: `make dev-up`

#### Frontend

The Next.js is viewable on <https://localhost:5173> on your browser of choice

#### Backend

The Hono server is assigned on port `3000`

### If you have no docker

#### Frontend

go to /frontend directory and just run `npm run dev` on your terminal

#### Backend

go to /backend directory and just run `npm run dev` on your terminal
